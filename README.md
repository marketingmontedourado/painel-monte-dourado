# Painel Monte Dourado

Dashboard de desempenho digital das marcas Monte Dourado Incorporações.

## Deploy

1. Fork este repositório
2. Conecte na Vercel (vercel.com)
3. Import o projeto — a Vercel detecta Vite automaticamente
4. Clique em Deploy

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Stack

- React 18 + Vite
- Recharts (gráficos)
- Lucide React (ícones)
- Fontes Marisa + Gotham (embutidas)
